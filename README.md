## Betül Halı Yıkama

Bandırma merkezli yerel bir halı yıkama işletmesi için Next.js App Router ile hazırlanmış üretime hazır tanıtım ve dönüşüm odaklı web sitesi.

## Teknolojiler

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Geliştirme

```bash
npm install
npm run dev
```

Uygulamayı `http://localhost:3000` adresinde açabilirsiniz.

## Üretim Build

```bash
npm run build
npm run start
```

## Düzenlenebilir İşletme Bilgileri

Tüm temel işletme verileri tek dosyada tutulur: `src/lib/site.ts`

## Klasör Yapısı

```text
src/
  app/
  components/site/
  lib/
```
